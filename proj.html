Java Backend

This is HTTP Server build in Java .It uses Socket.io and ServerSocket.io. This server is capable of handling more than one requests at a time by the virtue of mulit-threading. A Runnable class clientHandler servers mulitple users on the run time. It overrides the run() method to make the server work. This version currently supports only get requests to various files in the both html and png format

Hotel Booking and Managment

This is a MERN stack application that provides web based solutions for Hotel Management and bookings. You can register yourself here and put on the list of available Hotels and apartments. Customers can then come here, may register themselves or be guests and book your apartments from one of the Hotels. You will be notified as they book your apartments.

Shared Room Expense

Another MERN stack stack Application that is built for people living in a shared accomodataions it allows them to register there rooms and the roomates on the webiste. Each roomate through his/her separarte login can enter the room and put the purchase he made there along with the date and description and price.
